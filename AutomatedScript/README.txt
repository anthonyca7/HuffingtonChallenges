Created by Anthony Cabrera

Requirements:
	ruby 1.9.3

Steps:
	1) Run "bundle install" on this directory 
	2) Change the username and password on the visit_operations in the step_definitions folder,
	   make sure that huffingman post has permission to share an article.

	3) Run the automated task with "cucumber features/share_article.feature" on this directory

	If your internet connection is slow please uncomment the sleep lines in the visit_operation.rb file and re-run the application. I
	have tested this application on FireFox in Ubuntu 14.04


	It should work as expected, for any questions contact me at anthonyka7@gmail.com.
